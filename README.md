# 3-Bit Synchronous Counter

## Overview
Synthesizable RTL implementation of a 3-bit synchronous counter using Verilog HDL.

## Features
- Positive edge-triggered clock
- Synchronous reset
- Structured testbench verification

## Simulation
Waveform verified for correct binary count sequence.
